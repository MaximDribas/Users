# Техническое задание

Необходимо спроектировать и реализовать rest сервис хранения информации о
пользователях.

Стек: spring-core, spring-mvc, junit, mockito, по желанию использовать spring-boot
В качестве бд можно использовать h2 или свою inmemory реализацию.
наличие юнит и интеграционных тестов

Реализовать rest endpoints для создания пользователя, получение пользователя по id и
получения списка пользователей.
Формат передачи данных json.

Профиль пользователя состоит из полей:
login
password
firstName
lastName

Пример пользователя в формате json:
    {
   "userId": 1,

        "login": "login6char",

        "password": "password6char",

        "firstName": "First",

        "lastName": "Last"

    }
